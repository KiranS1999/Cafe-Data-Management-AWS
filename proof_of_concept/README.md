## Step 1
```
git clone -b dev git@github.com:delee3/team-4-project.git

```
## Step 2
```
Rename example.env to .env inside src directory

```
## Step 3
```
pip install -r requirements.txt

```
## Step 4 (Linux/Mac user)
```
chmod +x *.py

```
## Step 5
```
docker-compose up -d

```
## Step 6
Run python program from src directory
```
python start.py 

```
